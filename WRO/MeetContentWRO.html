What is content?

All information and experience
that impact users.

- Media - UX & UI - Technical documentation - e-learning & training materials

It is not text - if it was, we would call it text. Media = is an integral part of all the 3 above parts, icons, diagrams, instructional videos (short videos) Technical documentation = user guides, quick start guides, OLHs, so generally instructions on how to use a product e-Learning & training = instructor-led training, online training courses At MeetContent, we talk about all aspects of content as we just defined it, including: architecture, localization, (team) management, maintenance, tools, solutions, etc.
